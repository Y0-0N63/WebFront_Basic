기본 속성 선택자

태그에 작성된 특정 속성을 선택하는 선택자

  [작성법]
  선택자[속성명 = "속성값"] {
    css 코드;
  }

  (참고)
  1) 선택자 생략할 수 있음
  2) "속성자" 양쪽의 쌍따옴표(")는 홑따옴표(')로도 변경 가능

  [작성법]
  선택자1 > 선택자2 {
    css 코드;
